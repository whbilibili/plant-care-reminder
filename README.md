# Plant Care Reminder

家庭内部使用的植物养护提醒应用。

当前项目已按 harness 结构改造：

- `app/`：实际代码目录，当前保留为空白工作区，不在本次改造中修改
- `AGENTS.md`：Coding Agent 入口导航
- `ARCHITECTURE.md`：全局架构约束
- `PLANS.md`：项目级里程碑与长期规划
- `docs/`：产品文档、设计文档、执行计划、质量与安全约束

推荐技术方向：

- 前端：`React + Vite PWA`
- 数据与后端：`Convex`
- 目标设备：`iPhone`

后续如果开始编码，默认从 `docs/exec-plans/feature-list.json` 和 `docs/exec-plans/progress.txt` 进入，而不是重新扫描整个项目。
