# Security — 安全约束

> 本文件定义项目的安全红线和合规要求。P0 条目零容忍。

## 安全红线（P0）

- [ ] 禁止硬编码任何密钥、Token、密码
- [ ] 禁止将 Convex 管理密钥或其他服务端密钥暴露给客户端
- [ ] 禁止未鉴权用户写入家庭、植物、任务、订阅等数据
- [ ] 禁止跨家庭读取植物、任务和通知订阅
- [ ] 所有外部输入必须经过 Zod 或等效校验
- [ ] 图片上传能力必须绑定登录态和家庭权限校验

## 认证与授权规范

- 认证提供方：`Convex Auth` 或兼容邮件登录方案
- 权限边界：所有业务实体以 `familyId` 为隔离单元
- 写操作必须校验当前用户属于目标家庭空间

## 数据安全

- 所有时间统一以 UTC 存储
- 客户端只保存非敏感公开配置
- 推送订阅视为用户设备数据，禁止在 UI 中明文暴露完整 endpoint
