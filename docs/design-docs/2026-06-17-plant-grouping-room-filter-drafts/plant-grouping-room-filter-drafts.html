交互说明

视图切换器 (SegmentedControl)

  • 两个选项「全部」/「按房间」，紧凑样式
  • 位于搜索栏下方，与搜索栏等宽
  • 切换有 200ms fade 过渡动画
  • 用户选择持久化到 localStorage

分组头

  • 高度 40px，背景 --color-mist 半透明
  • 字体 14px 粗体，颜色 --color-ink
  • 数量标记用 --color-muted
  • 折叠/展开箭头 Chevron，16px
  • 点击分组头切换折叠/展开

房间筛选标签栏

  • 横向滚动，不换行，间距 8px
  • 选中：--color-leaf 背景 + 白色文字
  • 未选中：--color-mist 背景 + --color-ink 文字
  • 圆角 pill 形状 (999px)
  • 只展示有任务的 location

位置自动补全

  • 输入时下拉展示家庭已有 location 值
  • 匹配项高亮显示
  • 底部「+ 新建」入口允许自由输入
  • 数据来源：前端从植物列表去重

分组排序规则

  • 按分组内植物数量降序（多的在前）
  • 「未分组」永远排在最后
  • 组内排序与全部视图一致
  • 所有植物无 location 时只有「未分组」+ 引导文案

边界处理

  • location 名称过长时截断加省略号
  • 20+ 个 location 时分组全部展示（可折叠）
  • 搜索结果仍按分组展示
  • 新建植物设新 location 后自动出现新分组
