# Plant Care Reminder Moodboard

This moodboard qualifies the visual direction for the MVP mobile design.

## Creative Constraints

- Business: 家庭植物养护提醒 PWA
- Goal: 为 plant-care-reminder MVP 生成第一版移动端设计稿
- Audience: 家庭成员，主要使用 iPhone
- Visual language: 清爽园艺工具
- Palette: 叶绿 + 雾蓝 + 暖白
- Density: 手机紧凑 + 待办优先

## Recommended Direction

Use `01-task-first-dashboard.svg` as the primary product direction.

The first production design should prioritize:

- Today / overdue tasks as the first screen
- One-tap completion
- Plant photos as recognition anchors
- Quiet botanical color rather than decorative illustration
- Compact mobile controls for repeated use
